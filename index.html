部署到 GitHub Pages

  1. 在 GitHub 新建仓库（建议名：your-username.github.io）。
  2. 把本页保存为 index.html 放进仓库根目录并提交。
  3. 打开仓库 Settings → Pages，Build and deployment 选择 Branch: main / folder: /root。
  4. 几秒后访问：https://your-username.github.io/ 即可看到网站。
使用命令行（可选） 
git init
git add index.html
git commit -m "init site"
git branch -M main
git remote add origin https://github.com/<your-username>/<your-repo>.git
git push -u origin main
