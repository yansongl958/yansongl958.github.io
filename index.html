Deploy to GitHub Pages

  1. Create a new repository on GitHub (homepage mode: name it your-username.github.io).
  2. Save this page as index.html in the repository root and commit.
  3. Open Settings → Pages and set Build and deployment to Deploy from a branch, Branch: main, Folder: /(root).
  4. Wait a few seconds and visit: https://your-username.github.io/ (or /repo-name/ for project sites).
Using the command line (optional) 
git init
git add index.html
git commit -m "init site"
git branch -M main
git remote add origin https://github.com/<your-username>/<your-repo>.git
git push -u origin main
